# TDK ICM-45606 数据手册关键信息摘要

## 1. 设备概述

**来源：第1页**

ICM-45606是一款高性能6轴MEMS运动追踪设备，主要特性：
- 3轴陀螺仪 + 3轴加速度计
- 支持I3CSM、I2C和SPI串行通信接口
- 内置I2C主设备接口，可连接外部传感器
- 高达8KB的FIFO缓冲区
- 可编程中断
- 集成温度传感器

### 主要性能指标
| 参数 | 数值 |
|------|------|
| 陀螺仪噪声密度 | 3.8 mdps/√Hz |
| 加速度计噪声密度 | 70 µg/√Hz |
| 温度范围 | -40°C 至 +85°C |
| 供电电压(VDD) | 1.71V ~ 3.6V |

## 4. FIFO配置与使用

**来源：第79页, 第88-91页**

### 4.1 FIFO寄存器
| 地址 | 名称 | 功能 |
|------|------|------|
| 0x12-0x13 | FIFO_COUNT_0/1 | FIFO数据计数(字节数) |
| 0x14 | FIFO_DATA | FIFO数据读取端口 |

### 4.2 FIFO配置寄存器

#### FIFO_CONFIG0 (0x1D)
- **FIFO_MODE**: FIFO模式选择
- **FIFO_DEPTH**: FIFO深度配置

#### FIFO_CONFIG1_0/1 (0x1E-0x1F)
- **FIFO_WM_TH[15:0]**: 水印阈值设置

#### FIFO_CONFIG2 (0x20)
| 位 | 名称 | 功能 |
|----|------|------|
| 2 | FIFO_FLUSH | 写1清空FIFO |
| 1 | - | 保留 |
| 0 | FIFO_WM_GT_TH | 水印比较方向 |

#### FIFO_CONFIG3 (0x21)
| 位 | 名称 | 功能 |
|----|------|------|
| 5 | FIFO_HIRES_EN | 高分辨率模式 |
| 4 | FIFO_GYRO_EN | 陀螺仪数据入FIFO |
| 3 | FIFO_ACCEL_EN | 加速度计数据入FIFO |
| 2:1 | FIFO_ES1/ES0_EN | 外部传感器入FIFO |
| 0 | FIFO_IF_EN | 传感器寄存器写入FIFO |

#### FIFO_CONFIG4 (0x22)
| 位 | 名称 | 功能 |
|----|------|------|
| 5 | FIFO_COMP_EN | FIFO压缩使能 |
| 4 | FIFO_TMST_FSYNC_EN | 时间戳/FSYNC入FIFO |
| 0 | FIFO_ES0_6B_9B | 外部传感器0字节数选择 |

### 4.3 FIFO数据格式
- 默认支持16位数据格式
- 可配置为20位数据格式（用于高精度应用）
- 支持FIFO压缩算法，可存储4倍数据量

## 6. 间接寄存器访问(IREG)

**来源：第59-60页**

### 6.1 间接寄存器列表
某些寄存器需要通过IREG间接访问，包括：
- IMEM_SRAM区寄存器
- IMEM_SRAM_APEX区寄存器
- IMEM_SRAM_STC区寄存器
- IPREG_BAR区寄存器
- IPREG_TOP1区寄存器
- IPREG_SYS1区寄存器
- IPREG_SYS2区寄存器

### 6.2 访问方法

**间接寄存器地址 = 基础地址 + 目标寄存器偏移地址**

| 目标区域 | 基础地址 |
|----------|----------|
| IMEM_SRAM | 0x0000 |
| IMEM_SRAM_APEX | 0x0000 |
| IMEM_SRAM_STC | 0x0000 |
| IPREG_BAR | 0xA000 |
| IPREG_TOP1 | 0xA200 |
| IPREG_SYS1 | 0xA400 |
| IPREG_SYS2 | 0xA500 |

### 6.3 间接寄存器访问步骤

#### 写入步骤：
1. 设置IREG_ADDR_15_8 (0x7C) 和 IREG_ADDR_7_0 (0x7D)
2. 写入数据到 IREG_DATA (0x7E)
3. 等待最小间隔4µs
4. 等待IREG_DONE标志置位

#### 读取步骤：
1. 设置目标地址到 IREG_ADDR_15_8 和 IREG_ADDR_7_0
2. 等待CS=1(SPI)或STOP(I2C)触发预取
3. 等待最小间隔4µs
4. 读取 IREG_DATA 获取数据

### 6.4 关键IREG寄存器
| 地址(Hex) | 名称 | 功能 |
|-----------|------|------|
| 0x7C | IREG_ADDR_15_8 | 间接地址高字节 |
| 0x7D | IREG_ADDR_7_0 | 间接地址低字节 |
| 0x7E | IREG_DATA | 间接数据读/写端口 |
| 0x7F | REG_MISC2 | 包含SOFT_RST和IREG_DONE |

## 7. 数据获取方式

### 7.1 直接寄存器读取
通过I2C/SPI直接读取传感器数据寄存器：
```
ACCEL_DATA_X = (ACCEL_DATA_X1_UI << 8) | ACCEL_DATA_X0_UI
GYRO_DATA_X = (GYRO_DATA_X1_UI << 8) | GYRO_DATA_X0_UI
TEMP_DATA = (TEMP_DATA1_UI << 8) | TEMP_DATA0_UI
```

### 7.2 FIFO读取
1. 读取FIFO_COUNT获取有效数据字节数
2. 循环读取FIFO_DATA寄存器获取数据
3. 按FIFO数据包格式解析

### 7.3 中断驱动方式
1. 配置INT1/INT2中断
2. 使能数据就绪中断
3. 中断触发后读取数据

## 8. 数据格式与转换

### 8.1 加速度计数据
- 格式：16位有符号补码
- 温度转换公式：不适用（线性输出）

### 8.2 陀螺仪数据
- 格式：16位有符号补码
- 单位转换：根据量程计算

### 8.3 温度数据

**传感器寄存器格式（16位）：**
```
温度(°C) = TEMP_DATA / 128 + 25
```

**FIFO格式（8位）：**
```
温度(°C) = FIFO_TEMP_DATA / 2 + 25
```

## 9. 初始化流程

### 9.1 基本初始化步骤
1. **上电等待**：等待至少1ms
2. **读取WHO_AM_I**：验证设备ID (应为0x84)
3. **软复位**：设置REG_MISC2的SOFT_RST位
4. **等待稳定**：等待传感器稳定
5. **配置电源模式**：设置PWR_MGMT0
6. **配置传感器量程**：设置ACCEL_CONFIG0和GYRO_CONFIG0
7. **配置FIFO**（可选）：设置FIFO_CONFIG寄存器
8. **配置中断**（可选）：设置INT_CONFIG寄存器
9. **使能传感器**：根据需要使能加速度计/陀螺仪

### 9.2 推荐配置示例
```c
// 基本配置
PWR_MGMT0 = 0x0F;      // LN模式：陀螺仪+加速度计
ACCEL_CONFIG0 = 0x41;  // ±8g, 1.6kHz
GYRO_CONFIG0 = 0x41;   // ±1000°/s, 1.6kHz
```

## 11. 注意事项

1. **数据端序**：默认小端序，可通过SREG_CTRL寄存器修改为大端序
2. **间接访问限制**：不支持突发读写，需逐个访问
3. **最小时间间隔**：IREG访问间隔至少4µs
4. **SPI模式**：默认为Mode 1或Mode 3
5. **中断配置**：更改中断极性/模式需先禁用所有中断
6. **FIFO水印**：需要根据应用场景合理设置
