# ICM-45606 IMU验证任务 Handoff

## 任务背景

项目中原有 IMU 驱动支持 ICM-40607/40608/42670/42652 四款芯片（均属 ICM-42670 系列），新接入 **ICM-45606** 芯片。ICM-45606 采用全新的 IP 核设计，与旧系列**寄存器映射完全不同**。

---

## ICM-45606 vs 旧芯片 寄存器对比

### 芯片标识

| 芯片 | WHO_AM_I 地址 | WHO_AM_I 值 | 系列 |
|------|--------------|------------|------|
| ICM-45606 | **0x72** | **0x84** | 新 IP 核 |
| ICM-40607 | 0x75 | 0x38 | ICM-42670 系列 |
| ICM-40608 | 0x75 | 0x39 | ICM-42670 系列 |
| ICM-42670 | 0x75 | 0x67 | ICM-42670 系列 |
| ICM-42652 | 0x75 | 0x6F | ICM-42670 系列 |

### 关键寄存器地址对比

| 寄存器 | ICM-42670 系列 | ICM-45606 | 说明 |
|--------|---------------|-----------|------|
| WHO_AM_I | 0x75 | **0x72** | 地址和值都不同 |
| PWR_MGMT0 | 0x4E (40607) / 0x1F (42670) | **0x10** | Bank 0 低地址 |
| GYRO_CONFIG0 | 0x4F (40607) / 0x20 (42670) | **0x1C** | FS_SEL 位定义不同 |
| ACCEL_CONFIG0 | 0x50 (40607) / 0x21 (42670) | **0x1B** | 量程+ODR 用不同字段 |
| SOFTRESET | 0x11 (40607) / 0x02 (42670) | **0x7F (REG_MISC2)** | 完全不同的寄存器 |
| 数据区起始 | 0x1D (40607) / 0x09 (42670) | **0x00** | 从 ACCEL_DATA_X0 开始 |
| 0x51/0x52/0x53 | 使用（GYRO_CONFIG1 等） | **不存在** | ICM-45606 无这些寄存器 |

## ICM-45606 寄存器详细位定义

### PWR_MGMT0 (0x10)

| 位 | 名称 | 功能 |
|----|------|------|
| 7:4 | - | 保留 |
| 3:2 | GYRO_MODE | 00=关闭, 01=待机, 10=低功耗, **11=低噪声** |
| 1:0 | ACCEL_MODE | 00=关闭, 01=关闭, 10=低功耗, **11=低噪声** |

**低噪声模式**: PWR_MGMT0 = 0x0C | 0x03 = **0x0F**

### GYRO_CONFIG0 (0x1C)

| 位 | 名称 | 值 | 功能 |
|----|------|-----|------|
| 7:4 | GYRO_UI_FS_SEL | **0011** | ±500 dps |
| 3:0 | GYRO_ODR | **0110** | 800 Hz |

**配置值**: 0x30 | 0x06 = **0x36**

### ACCEL_CONFIG0 (0x1B)

| 位 | 名称 | 值 | 功能 |
|----|------|-----|------|
| 6:4 | ACCEL_UI_FS_SEL | **001** | ±16 g |
| 3:0 | ACCEL_ODR | **0110** | 800 Hz |

**配置值**: 0x10 | 0x06 = **0x16**

### REG_MISC2 (0x7F) — 软复位

| 位 | 名称 | 功能 |
|----|------|------|
| 0 | SOFT_RST | 写 1 触发软复位，**自动清零** |
| 7:1 | - | 保留 |

写入值: **0x01**，复位后硬件自动清零回到 0x00。

## 数据格式差异（关键）

### 数据寄存器顺序

ICM-45606 从 0x00 地址连续读取 14 字节，顺序与旧芯片**不同**：

| 偏移 | ICM-45606 | ICM-42670 系列 |
|------|-----------|---------------|
| 0-1 | **ACCEL_DATA_X** | TEMP_DATA |
| 2-3 | **ACCEL_DATA_Y** | ACCEL_DATA_X |
| 4-5 | **ACCEL_DATA_Z** | ACCEL_DATA_Y |
| 6-7 | **GYRO_DATA_X** | ACCEL_DATA_Z |
| 8-9 | **GYRO_DATA_Y** | GYRO_DATA_X |
| 10-11 | **GYRO_DATA_Z** | GYRO_DATA_Y |
| 12-13 | **TEMP_DATA** | GYRO_DATA_Z |

> **注意**: ICM-45606 温度在最后两个字节，旧芯片在最前面两个字节。

### 数据字节序

| 项目 | ICM-42670 系列 | ICM-45606 默认 | ICM-45606 配置后 |
|------|---------------|---------------|-----------------|
| 字节序 | **大端**（高字节在前） | 小端（低字节在前） | **大端**（通过 SREG_CTRL 切换） |

ICM-45606 默认小端序，需配置 `SREG_CTRL[SREG_DATA_ENDIAN_SEL] = 1` 切换为大端序以匹配旧芯片架构。

### 温度转换公式

| 芯片 | 公式 | 除数 |
|------|------|------|
| ICM-45606 | `T(°C) = TEMP_DATA / 128.0 + 25` | 128.0 |
| ICM-42670 系列 | `T(°C) = TEMP_DATA / 132.48 + 25` | 132.48 |

> ICM-45606 使用 2 的幂除数（128），便于定点运算；旧芯片使用非整数除数（132.48）以获得更精确的温补。

## SREG_CTRL 大端序配置（IREG 间接访问）

### 为什么需要间接访问

`SREG_CTRL` 寄存器位于 `IPREG_TOP1` 地址空间（基地址 0xA200，偏移 0x67），**不在 User Bank 0 的直接地址范围内**，必须通过 IREG（Indirect Register Access）机制访问。

### IREG 寄存器（User Bank 0）

| 地址 | 寄存器 | 功能 |
|------|--------|------|
| 0x7C | IREG_ADDR_15_8 | 间接地址高字节 |
| 0x7D | IREG_ADDR_7_0 | 间接地址低字节 |
| 0x7E | IREG_DATA | 间接数据端口 |

### 目标地址计算

```
SREG_CTRL 间接地址 = IPREG_TOP1_BASE + SREG_CTRL_OFFSET
                   = 0xA200 + 0x67
                   = 0xA267
```

### 写入流程（关键约束）

```
IREG 写入 = 0x7C←0xA2, 0x7D←0x67, 0x7E←0x02
```

| 步骤 | 寄存器 | 写入值 | 说明 |
|------|--------|--------|------|
| 1 | IREG_ADDR_15_8 (0x7C) | 0xA2 | 间接地址[15:8] |
| 2 | IREG_ADDR_7_0 (0x7D) | 0x67 | 间接地址[7:0] |
| 3 | IREG_DATA (0x7E) | 0x02 | bit1=1 → 大端序使能 |

> **硬约束**：数据手册要求 IREG 写入**必须**在单次 SPI 突发写事务中完成（CS 保持低电平，3 字节连续写入地址 0x7C→0x7D→0x7E）。使用独立的三次 `WriteOneByte`（CS 每次拉高）会导致 IREG 地址锁存器在 CS 上升沿被清空，写入无效。

### 验证方法

写入后通过 IREG 读回 SREG_CTRL 值：
```
IREG 读 = 0x7C←0xA2, 0x7D←0x67 → 读 0x7E
期望: 0x02
```

## 认知陷阱

1. **禁止假设新芯片寄存器兼容旧芯片** — ICM-45606 与 ICM-42670 系列寄存器映射完全不同
2. **禁止忽略字节序差异** — ICM-45606 默认小端序，需通过 SREG_CTRL 配置切换
3. **禁止向不存在的寄存器写入** — 0x51/0x52/0x53 在 ICM-45606 中不存在
4. **IREG 写入必须突发** — 三次独立 `WriteOneByte` 无效，CS 拉高会清空地址锁存
5. **温度是字节序的最佳哨兵** — 温度异常（-199°C / 89°C）时优先检查字节序
